BeeWare¶

编写 Python。在任何地方运行。

BeeWare 是一套工具和库，每一个都能帮助你编写跨平台的本地 GUI Python 应用程序。它包括：

  • Toga，一个跨平台的小部件工具包；

  • Briefcase，将 Python 项目打包成可分发的人工制品的工具，可以运送给终端用户；

  • Rubicon ObjC, a library for working with Objective C libraries on iOS and macOS from within Python code;

  • Rubicon Java, a library for working with Java libraries from within Python code;

  • 预先编译好的 Python 构建，可以在没有官方 Python 安装程序的平台上使用。

你可以孤立地使用每一个工具，也可以把它们作为一个套件来使用。

完整的 BeeWare 套件还包括 软件开发工具 和 应用程序，这些都是使用 BeeWare 自己的库编写的。

BeeWare 套件可在 macOS、Windows、Linux（使用 GTK）；Android 和 iOS 等移动平台上使用；也可用于网络。对其他平台（如机顶盒和手表）的支持已列入我们的长期路线图。

平台支持

BeeWare 套件是一项正在进行的工作。一些工具和库比其他的更成熟，一些平台比其他平台有更好的支持。特别是，网络支持应该被认为只是试验性的。

准备好亲自尝试一下 BeeWare 了吗？ 让我们用 Python 构建一个跨平台的应用程序吧!
